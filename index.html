Course Outline

Module 1 – Core Architecture

Game loop, delta timing, ECS basics. Learn how to structure your game from the ground up using solid architectural patterns.

Module 2 – Graphics & Rendering

Sprites, layers, 2D camera with MonoGame. Master rendering pipelines and drawing performance optimizations.

Module 3 – Collision & Physics

Bounding boxes, movement, physics. Simulate 2D worlds with reliable collision detection and physics rules.

Module 4 – Audio, UI & Input

Sound, menus, input systems. Build immersive game interfaces and responsive player controls.

Module 5 – Tools & Level Editor

Build in-engine editors and tools. Speed up your workflow and support level design within your engine.

Module 6 – Capstone Project

Create your own mini 2D game engine. Combine all knowledge gained into a real-world project.
